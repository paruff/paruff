**The uFawkes family** — composable Docker stacks that implement the DORA AI Capabilities Model:

| Stack | What it does | Status |
|---|---|---|
| [🦅 Fawkes](https://github.com/paruff/fawkes) | Full Kubernetes IDP — Backstage + ArgoCD + 6 DORA metrics | Active |
| [🤖 uFawkesAI](https://github.com/paruff/uFawkesAI) | AGENTS.md template for all AI coding agents | Active |
| [📊 uFawkesObs](https://github.com/paruff/uFawkesObs) | Prometheus + Grafana + AI observability | Active |
| [🔧 uFawkesPipe](https://github.com/paruff/uFawkesPipe) | Polyglot CI/CD pipeline contract | Active |
| [📈 uFawkesDORA](https://github.com/paruff/uFawkesDORA) | 6 DORA metrics from GitHub data | In progress |
| [🛠️ uFawkesDevX](https://github.com/paruff/uFawkesDevX) | Developer control plane | In progress |
